# Pharmacy Inventory Management System

## Overview

The Pharmacy Inventory Management System is a standalone desktop application developed to streamline pharmacy operations by managing users, authentication, and inventory processes efficiently.
The system follows Rapid Application Development principles, focusing on fast prototyping, iterative development, and early functional delivery.

This project is built using **C#** on the **Visual Studio** platform, with UI designs initially prototyped using **Figma** to ensure a clear and user friendly interface.

---

## Technologies Used

* **Programming Language**: C#
* **Framework**: .NET, GunaUI
* **IDE**: Visual Studio
* **Database**: SQL-based local database
* **UI Designing**: Figma
